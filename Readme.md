Papt — это свободно распространяемая обёртка для пакетного менеджера pacman (используемого в дистрибутивах Arch Linux и производных) с синтаксисом, похожим на пакетный менеджер apt (используемый в дистрибутивах Debian и производных), **без** AUR Helper (Arch User Repository Helper).

> Если вам нужен Papt с AUR Helper, используйте проект [Papt2](https://github.com/MarkGMX8X/papt2).

> Если вам нужен Papt с полным синтаксисом как у apt используйте проект [Papt3](https://github.com/MarkGMX8X/papt3)

> ⚠️ **Важно:** Этот пакет устанавливает команду **`papt`**.  
> Если у вас уже установлен `papt2` или `papt3` — удалите их перед установкой `papt`:
> ```bash
> sudo pacman -Rns papt2    # или papt3
> ```

> Стрница на [AUR](https://aur.archlinux.org/packages/papt)

## Примеры использования

| Команда | Действие |
|---------|----------|
| `papt install <пакет>` | Установка из официальных репозиториев |
| `papt install -r <пакет>` | Альтернатива (официальные репозитории) |
| `papt install -p <пакет>` | Альтернатива 2 (официальные репозитории) |
| `papt remove <пакет>` | Удаление пакета(ов) |
| `papt update` | Обновление баз репозиториев |
| `papt update -Sys` | Полное обновление системы (базы + пакеты) |
| `papt search <пакет>` | Поиск пакета(ов) |

## Установка

Требуемые пакеты: git, base-devel
```bash
git clone https://github.com/MarkGMX8X/papt.git
cd papt
sudo cp papt /usr/local/bin/
```

## Установка с AUR (Ручная)
```bash
git clone https://aur.archlinux.org/papt.git
cd papt
makepkg -si
```

## Установка с AUR (Через AUR Helper)
```bash
yay (Или другой AUR Helper) -S papt
```
